# 🚔 Template Landing Page – Estilo GCM

Landing page institucional moderna construída com React, TypeScript, Vite e TailwindCSS, inspirada em interfaces de segurança pública (GCM).

O projeto foi desenvolvido com foco em arquitetura limpa, escalabilidade e boas práticas de UI/UX.

---

## ✨ Funcionalidades

* ⚡ Setup moderno com Vite
* 📱 Layout totalmente responsivo
* 🎨 Estilização com TailwindCSS
* 🧠 Arquitetura organizada (components, layouts, pages)
* 🔀 Navegação com React Router
* 📰 Página de notícias com destaque + lista
* 📊 Seção de estatísticas (transparência)
* 🎠 Carrossel com cards visuais
* 📞 Seção de contato com ícones e alinhamento
* ✨ Microinterações e efeitos hover
* 🧭 Destaque de rota ativa na navbar

---
